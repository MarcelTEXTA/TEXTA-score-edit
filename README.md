# TEXTA score edit (fr)
TEXTA score edit est un éditeur de partitions musicales codées en C#. Il permet aux utilisateurs de créer, modifier et visualiser des partitions musicales de manière intuitive. Il offre une interface conviviale avec des fonctionnalités basiques pour le moment mais qui seront enrichies dans les futures versions.
## Fonctionnalités principales
- Création de partitions musicales.
- Édition de notes et de rythmes.
- Visualisation en temps réel des modifications apportées.
- Sauvegarde et chargement de fichiers de partitions.
- Convertisseur de fichiers MIDI et audios en partitions.
## Installation
1. Cloner le dépôt GitHub pour obtenir le code source :
   ```bash
   git clone https://github.com/MarcelTEXTA/TEXTA-score-edit.git
    ```
L'appliaation est disponible mais possède beaucoup de bugs, les fonctionnalités ne sont pas encore complètes.
[TEXTA score edit v0.1.0 DevPreview](https://github.com/MarcelTEXTA/TEXTA-score-edit/raw/refs/heads/main/TEXTA_score_edit/publish/setup.exe)

# TEXTA score edit (en)
TEXTA score edit is a C# coded music score editor. It allows users to create, edit, and visualize music scores intuitively. It offers a user-friendly interface with basic features for now but will be enriched in future versions.
## Main Features
- Creation of music scores.
- Editing of notes and rhythms.
- Real-time visualization of modifications.
- Saving and loading score files.
- MIDI and audio file converter to scores.
## Installation
1. Clone the GitHub repository to get the source code:
   ```bash
   git clone https://github.com/MarcelTEXTA/TEXTA-score-edit.git
    ```
The application is available but has many bugs, and the features are not yet complete.
